# ThumbnailMaker — Frontend

React 19 + Vite + Tailwind v4 frontend for **ThumbnailMaker**.

See the [root README](../README.md) for architecture, setup, and full project docs.

## Quick start

```bash
npm install
cp .env.example .env   # set VITE_API_BASE_URL if backend runs elsewhere
npm run dev
```

Dev server runs at `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```
